# Metode_Bagi_Dua-f-x-x-3-x-2-2x-1-0-01-
Implementasi Metode Bagi Dua (Bisection Method) untuk mencari akar persamaan nonlinear f(x)=x^3+x^2-2x-1 dengan pendekatan iteratif hingga error 0,01%. Proses dilakukan hingga konvergen menggunakan analisis iterasi dan error relatif sebagai dasar penghentian.
